# js的this绑定问题

什么是this

内部或者外部的函数环境的不同，this的值会发生改变

+ 默认绑定

+ 隐性绑定

+ 显性绑定

  ​	call(对象,调用函数的参数) 

  ​	apply(d对象,[参数；列表])

+ new 绑定

  ​	1 、创建一个新对象

  ​	2 、把新对象的proto属性指向原函数的prototype属性

  ​	3 、将新对象绑定到这个函数的this上

  ​	4、返回新对象

  ### 绑定的优先级别

  new >显示绑定 >隐性绑定 >默认绑定
